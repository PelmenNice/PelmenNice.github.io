Чтобы запустить сервер, нужно запустить в папке терминал и ввести команду:
python -m http.server 80
Эта команда запустит сервер, и любой компьютер в локальной сети сможет увидеть сайт, введя IP-адрес комьютера и порт сервера в браузер.
Например: 192.168.56.51:8000
Если команда выдает ошибку, стоит попробовать сменить число в конце команды (порт сервера).
